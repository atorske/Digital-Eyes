Project Overview

'Digital Eyes' is an application designed for the Windows 7 Phone. It intends to help the visually disbled navigate inside of buildings.

Features:



  • Voice Recognition: Allows users to issue verbal commands to the app.
  • Verbal Navigation: Application gives directions vocally.
  • Microsoft Tag Recognition: Navication based on free Microsoft Tags.
  • On-Screen Map: Directions also shown on the phone's screen.

Users:


The target user is a visually disabled person. The application is designed to improve the mobility and navigation within buildings for the visually disabled. However, though the application is targeted at the visually disabled it will also be extremely usable by the sighted. Verbal navigational cues can be used by anyone, disabled or not. The total user base is anyone who might get lost inside of buildings and need an application similar to a car gps system, except for inside of buildings.
