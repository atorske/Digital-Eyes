Voice Recognition

One key feature of our application is voice recognition. Since the target user is visually disabled an application with solely an on-screen map will not suffice. We plan to have the application follow along on a map onscreen as the user moves, however, the applicatin will communicate via vocal commands from the user. The application will also resond with audible output, in the form of current location, and directions to desired destination.

This functionality will be greatly helped by the fact that the Windows Phone has a built in voice recognition hardware, in fact one of the best on the market.
